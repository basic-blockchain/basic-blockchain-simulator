# Release v0.14.0

## Summary

Phase I.5 focuses on enriched admin capabilities. This release expands
RBAC permissions and adds new admin endpoints for soft-delete/restore,
profile edits, and wallet management (freeze/unfreeze, list wallets).

## Included Changes

- **Admin API.** New endpoints and repository support for user soft-delete,
  restore, profile edits, and listing all wallets.
- **RBAC.** New permissions for delete/restore plus explicit audit actions;
  ADMIN baseline now includes `VIEW_WALLETS`, `FREEZE_WALLET`, and
  `UNFREEZE_WALLET`.
- **Persistence.** Repository layer adds soft-delete, profile edit, and
  list-all-wallets capabilities.

## Validation

- Not run for this release. Use the standard test suite from the prior
  release pipeline.

## Migration / breaking notes

- New permissions must be present for admin actions. Existing ADMIN users
  get the additional baseline permissions automatically.

## Notes

- This release pairs with the frontend v0.8.0 admin enrichment UI.
